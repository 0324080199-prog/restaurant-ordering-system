# 🎯 Here's how we'll build Milestone 1

### Step 1 — Loading Screen

Show a branded loading screen while the website loads.

### Step 2 — Live Search

Users type and the menu filters instantly.

### Step 3 — Category Filter

Buttons like:

- All
- Burgers
- Pizza
- Rice
- Drinks
- Desserts

### Step 4 — Favorite Meals ❤️

Users can save meals, and favorites persist with Local Storage.

### Step 5 — Order History

Store completed orders in Local Storage so users can view previous receipts.

### Step 6 — Customer Reviews

Display testimonials or sample reviews with star ratings. Later, when you add a backend, these can become real user-submitted reviews.
